# StudyStreak 🔥

A full-stack habit tracker built to help students stay consistent with daily goals — track study habits, build streaks, and visualize weekly progress.

## 🌐 Live Demo

Frontend:
https://studystreak-cyan.vercel.app

Backend API:
https://studystreak-backend-plby.onrender.com

---

## ✨ Features

- Secure JWT Authentication
- User Signup & Login
- Create daily habits
- Edit habits
- Delete habits
- Mark habits as completed
- Automatic streak tracking
- Weekly analytics dashboard
- GitHub-style contribution heatmap
- Responsive modern UI
- MongoDB Atlas cloud database

## 🚀 Running Locally

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Create a `.env` file inside backend:

```
MONGO_URI=your_connection_string
JWT_SECRET=your_secret
PORT=5000
```
